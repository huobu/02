### 信息  
* **一切都是信息**
* 香农：信息是用来消除随机不确定性的东西。
* 维纳：信息是人们在适应外部世界，并使这种适应反作用于外部世界的过程中，同外部世界进行互相交换的内容和名称。

  信息本身的机密性（Confidentiality）、完整性（Integrity）和可用性（Availability），即防止未经授权使用信息、防止对信息的非法修改和破坏、确保及时可靠地使用信息。
